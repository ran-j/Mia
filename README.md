<h2 align="center">Mia</h2>

A virtual assistant make in VB.net.

# Run

* Microsoft Visual Studio 17 or above.
* Microsoft .Net 4.6 or abouve is required.

# Features

* Voice command
* PS1 emulator
* Net tools
* Control Arduino
* Spotfy manager
* Control Volume
* System monitoring

And more
